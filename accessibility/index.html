Inclusive Design

Care without Boundaries

At Cross Care Group, we are committed to ensuring our digital experiences are accessible to everyone, regardless of ability or technology.

Our Commitment

We believe that information and healthcare support should be available to all. We are actively working to increase the accessibility and usability of our website and in doing so adhere to many of the available standards and guidelines.

WCAG 2.1 Standard

We strive to conform to the World Wide Web Consortium (W3C) Web Content Accessibility Guidelines 2.1 Level AA.

Continuous Testing

Our platform is frequently audited using both automated tools and manual testing with assistive technologies.

User Feedback

We actively seek and act upon feedback from users with diverse needs to improve our interface.

Navigation & Tools

Key Features

Visual Access

  • High contrast color schemes for text and interface elements.
  • Support for text resizing and browser zooming up to 200%.
  • Descriptive alt-text for all informative images.

Functional Access

  • Logical tab order and visible keyboard focus states.
  • Skip-to-content links for efficient navigation.
  • Semantic HTML structure (H1-H4) for screen reader clarity.
